# 大话强化学习-代码实现
本仓库是是强化学习基本算法的一个实现，并用网格世界作为展示，包含网格世界的可视化、策略的可视化，帮助学习强化学习。
# 代码说明
**grid.py** <br>
agent所在环境，网格世界的实现，可以配置网格的大小，边界，reward奖励，终点，禁止区域，所有的都可以可视化。
![网格世界-1](example/grid_1.png)
![网格世界-2](example/grid_2.png)
红色网格是禁止区域，黄色网格是终点 <br>
